# NemaFacilityLocationProblem

Solves a multi-year facility location problem.
This code is used as part of the Business Strategy example for Nema, the data management platform.

## Inputs

- Existing and proposed facility locations, along with their maximum yearly capacity, and their monthly lease costs.
- Yearly demand and locations of customers.
- Company information, mostly assumed discount rate to compute Net Present Value (NPV)

## Outputs

- Plan for which facilities to use, along with their projected yearly volumes.
- Breakdown of infrastructure and transportation costs per year.
